Kinesiolgo

Mi proyecto final en desarrollo web fue un sitio web profesional para un kinesiologo.

En esta primera preentrega, decidi crear un algoritmo para que el usuario pueda hacer un pequeño test sobre habitos y ergonomia en el trabajo.

A traves de este test se busca por una parte, que el usuario pueda tomar nocion sobre sus habitos a mejorar.

Por otra parte, el profesional puede obtener una evaluacion rapida sobre habitos del paciente y ademas, mostrar la necesidad de sus servicios a su posibles pacientes.

Este algoritmo se ampliara eventualmente con otros pequeños test, apuntando a otras profesiones que no requieren de muchas horas sentados, como trabajos repetitivos y trabajos pesados.
